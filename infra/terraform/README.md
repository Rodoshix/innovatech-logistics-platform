# Terraform

Esta carpeta contiene la infraestructura AWS definida como código.

Contenido:

- Red base: VPC, subred pública y subred privada.
- Security Groups para separar tráfico público y privado.
- Recursos de cómputo para ejecutar los servicios de la plataforma.
- Amazon ECR para registro de imágenes.
- Amazon ECS para orquestación de contenedores.
- Outputs necesarios para documentar IPs, endpoints y recursos creados.
